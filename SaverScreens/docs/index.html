Overview »
The purpose of SaverScreens is to extend the functionality of the SaverBeans SDK to make it easier to create screensavers in Java. A SaverScreens screensaver may be either 2D-, 3D-, or Panel-based (i.e. uses a LayoutManager), with each of the base classes designed to simplify the construction of each type.

SaverScreens also includes various utility classes such as a Screensaver Applet (used to demonstrate a screensaver in an applet) and a Screensaver Runner (executes a screensaver without any platform dependencies). Some sample screensavers are also included to demonstrate SaverScreens functionality.
